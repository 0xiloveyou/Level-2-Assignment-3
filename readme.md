# Football Ticket Booking System - Database Design & SQL Queries

## Overview

This project implements a simplified **Football Ticket Booking System** using PostgreSQL. The database manages football fans, tournament matches, and ticket booking transactions.

The project demonstrates:

- Database design and ERD modeling
- Primary Keys and Foreign Keys
- Referential Integrity
- One-to-Many and Many-to-One Relationships
- Data validation using CHECK constraints
- Intermediate and Advanced SQL queries
- Joins, Subqueries, Aggregations, Pattern Matching, NULL handling, and Pagination
